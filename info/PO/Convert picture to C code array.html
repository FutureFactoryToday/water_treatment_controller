This tool is converting your .gif/.jpg/.jpeg/.png image file to embedded C/C++ code style array or string: {HEX: \x..,0x..} and binary file for download, the data format is compatible for all Digole serial modules.
You can resize the image to fit your requirement, if you only input "width" or "height", the software will calculate other side automatically, if you input both fields, the software will strech the shorter side.
